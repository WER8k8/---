
**需要修改的变量**（4个）:
1. `DB_PASSWORD` - 数据库密码
2. `MINIO_SECRET_KEY` - MinIO秘密密钥
3. `JWT_SECRET_KEY` - JWT签名密钥
4. `AI_NVIDIA_API_KEY` - NVIDIA AI API密钥

**修复方法**:
```bash
# 生成安全密钥
python -c "import secrets; print(secrets.token_hex(32))"

# 编辑 .env.prod 文件，替换以下占位符：
# DB_PASSWORD=your-secure-db-password-change-in-production
# MINIO_SECRET_KEY=your-minio-secret-key-change-in-production
# JWT_SECRET_KEY=your-32-character-or-longer-secret-key-must-change-in-production
# AI_NVIDIA_API_KEY=your-nvidia-api-key
```

# AI 智能体系统配置

## 智能体架构

本系统配置了 46 个专业智能体，采用分层架构协同工作：

```
超级智能体指挥官 (agent-commander)
        │
   ┌────┴──────────────────────────────────────────┐
   │  多智能体协作编排器 (multi-agent-orchestrator)   │
   │     并行任务执行 / 负载均衡 / 分布式计算          │
   └────┬──────────────────────────────────────────┘
        │
   ┌────┼────────────────────┬────────────────┐
   │    │                    │                │
   ▼    ▼                    ▼                ▼
开发层  测试层              DevOps层           AI层
(20个)  (3个)               (4个)             (5个)
        │                    │                │
        └────────────────────┴────────────────┘
                        │
                  全周期支持层
          (自适应思考 / 自动修复 / 根因分析
           暴力算力 / MCP工具创建 / 自我进化)
```

| **backend-page-builder** | vue-vben-admin | 后台基础框架集成 |
| **git-expert** | Git, GitHub | 仓库管理 + 分支策略 |

## 全周期支持智能体

- **adaptive-thinking**: 复杂问题时自动加深推理深度
- **auto-fix**: 同类错误出现3次以上自动介入修复
- **compute-booster**: 大模型卡顿时反向调用云端GPU算力
- **root-cause-analysis**: 疑难Bug深度根因分析
- **mcp-tool-creator**: 按需创建新的MCP工具扩展能力
- **self-improvement**: 每阶段结束后自动复盘优化
- **skills-search**: 持续发现新技能并补充

## 混合开发模式

```
模式A: InsCode (快马) 生成 → vue-vben-admin SEO后台UI
模式B: 手写核心代码 → FastAPI + AI引擎 + 复杂业务逻辑
模式C: GitHub开源SEO工具 → RankReady / GEO AI 等集成
模式D: MCP工具扩展 → mcp-tool-creator 按需创建
```

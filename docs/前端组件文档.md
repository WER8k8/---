# 保温建材独立站 - 前端组件文档

本文档介绍了保温建材独立站展示管理模块的核心组件、SEO 优化、响应式设计和性能优化功能。

## 📦 核心组件

### 1. ProductManager - 产品管理组件

**位置**: `src/components/ProductManager.tsx`

**功能**:
- 产品列表展示（支持分类筛选和搜索）
- 产品 CRUD 操作（增删改查）
- 图片 URL 管理
- 评分和价格管理
- 标签管理

**使用示例**:
```vue
<ProductManager
  :products="products"
  @products-change="handleProductsChange"
/>
```

**Props**:
| 属性 | 类型 | 说明 |
|------|------|------|
| products | Product[] | 产品列表数据 |

**Events**:
| 事件 | 参数 | 说明 |
|------|------|------|
| products-change | newProducts: Product[] | 产品列表变更 |

## 🔍 SEO 组件

### SEOHead - SEO 元数据组件

**位置**: `frontend/components/SEOHead.vue`

**功能**:
- 动态页面标题和描述
- Open Graph 标签（Facebook、LinkedIn 等社交平台）
- Twitter Card 支持
- Canonical URL
- 结构化数据（JSON-LD）

**使用示例**:
```vue
<SEOHead
  title="页面标题"
  description="页面描述"
  :keywords="['关键词1', '关键词2']"
  og-type="website"
  og-image="/images/og.jpg"
  twitter-card="summary_large_image"
  canonical-url="https://example.com/page"
  :structured-data="structuredData"
/>
```

**Props**:
| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| title | string | '' | 页面标题 |
| description | string | '' | 页面描述 |
| keywords | string[] | [] | 关键词列表 |
| ogType | 'website' \| 'article' \| 'product' | 'website' | Open Graph 类型 |
| ogImage | string | '' | OG 图片 URL |
| twitterCard | 'summary' \| 'summary_large_image' | 'summary_large_image' | Twitter Card 类型 |
| canonicalUrl | string | '' |  canonical URL |
| structuredData | object \| object[] | undefined | JSON-LD 结构化数据 |

---

### useSEOHead - SEO 工具函数

**位置**: `frontend/composables/useSEOHead.ts`

**提供的函数**:

```typescript
// 生成组织结构化数据
generateOrganizationSchema()

// 生成产品结构化数据
generateProductSchema(product: Product)

// 生成文章结构化数据
generateArticleSchema(article: {...})

// 生成案例结构化数据
generateCaseSchema(caseStudy: Case)

// 生成分页导航结构化数据
generateBreadcrumbSchema(items: Array<{name, url}>)

// 生成 FAQ 结构化数据
generateFAQSchema(questions: Array<{question, answer}>)
```

**使用示例**:
```typescript
import { useSEOHead } from '~/composables/useSEOHead'

const { generateProductSchema, generateBreadcrumbSchema } = useSEOHead()

const structuredData = [
  generateProductSchema(product),
  generateBreadcrumbSchema([
    { name: '首页', url: '/' },
    { name: '产品中心', url: '/products' },
    { name: product.name, url: `/products/${product.id}` }
  ])
]
```

## 📱 响应式设计组件

### useResponsive - 响应式工具 Composable

**位置**: `frontend/composables/useResponsive.ts`

**功能**:
- 断点检测（xs, sm, md, lg, xl, 2xl）
- 设备类型判断（mobile, tablet, desktop）
- 触摸设备检测
- 响应式类名生成

**使用示例**:
```typescript
import { useResponsive } from '~/composables/useResponsive'

const {
  isMobile,
  isTablet,
  isDesktop,
  currentBreakpoint,
  responsiveClass,
  getGridColumns
} = useResponsive()

// 获取响应式网格类
const gridClass = getGridColumns({ xs: 1, sm: 2, md: 3, lg: 4 })
```

**断点配置**:
| 断点 | 最小宽度 | 设备类型 |
|------|----------|----------|
| xs | 0px | 手机竖屏 |
| sm | 640px | 手机横屏 |
| md | 768px | 平板 |
| lg | 1024px | 小屏幕桌面 |
| xl | 1280px | 大屏幕桌面 |
| 2xl | 1536px | 超大屏幕 |

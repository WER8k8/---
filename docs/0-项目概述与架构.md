# 轻集料混凝土官网 — 项目概述与技术架构

## 项目定位
企业级轻集料混凝土官方网站，融合 AI SEO 智能运营系统，实现百度权重10目标。采用**混合开发模式**：InsCode（快马）生成后台UI + 手写核心业务代码 + GitHub开源AI SEO工具集成。

## 技术架构总览

```
用户访问 (Browser/Crawler)
        │
   ┌────▼────┐
   │  CDN    │  (Cloudflare/阿里云CDN)
   └────┬────┘
        │
   ┌────▼─────────────────────────────────┐
   │  Nuxt.js 3 前端 (SSR 服务端渲染)      │
   │  - Vue 3 + TypeScript + Tailwind     │
   │  - Pinia 状态管理                    │
   │  - vue-vben-admin 后台管理            │
   │  - Shadcn/ui 组件库                  │
   └────┬─────────────────────────────────┘
        │ HTTP/REST API
   ┌────▼─────────────────────────────────┐
   │  FastAPI 后端 (Python 3.11)           │
   │  - SQLAlchemy ORM                    │
   │  - Pydantic 数据验证                  │
   │  - Celery 异步任务队列                │
   │  - JWT 认证                          │
   └────┬────┬──────────────────────┬──────┘
        │    │                      │
   ┌────▼┐ ┌▼────────┐  ┌─────────▼──────┐
   │Postgre│ │Redis   │  │  AI 引擎        │
   │SQL   │ │ 缓存   │  │  LangChain      │
   │15.6  │ │7.2.4   │  │  LlamaIndex     │
   └──────┘ └────────┘  │  OpenAI/Claude  │
                        └────────┬────────┘
                                 │
                        ┌────────▼────────┐
                        │  Crawler 引擎    │
                        │  Playwright      │
                        │  BeautifulSoup   │
                        └─────────────────┘
```

## 核心业务模块
| 模块 | 技术实现 | 阶段 |
|------|---------|------|
| 产品展示 | Nuxt.js + FastAPI CRUD | Phase 1 |
| 内容管理 | FastAPI + PostgreSQL | Phase 1 |
| SEO仪表盘 | vue-vben-admin + ECharts | Phase 2 |
| AI内容优化 | LangChain + LLM API | Phase 2 |
| LLMs.txt生成 | FastAPI + AI Engine | Phase 2 |
| 网站审计 | Playwright + NLP | Phase 2 |
| Schema标记 | FastAPI + JSON-LD | Phase 3 |
| EEAT信号 | AI Engine + 结构化数据 | Phase 3 |
| GitHub SEO | GitHub Actions + 自动优化 | Phase 3 |
| 合规审查 | NLP + 规则引擎 | Phase 3 |
| 权限管理 | JWT + RBAC | Phase 3 |

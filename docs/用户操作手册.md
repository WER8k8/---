## 2. 登录与权限

### 2.1 登录系统

1. 访问后台地址: `https://youding.com/admin`
2. 输入用户名和密码
3. 点击"登录"按钮

**默认账号**（首次部署后请修改密码）:
- 用户名: `admin`
- 密码: `admin123`

### 2.2 权限说明

系统采用RBAC三层权限模型：

| 角色 | 权限范围 | 适用人员 |
|------|---------|---------|
| 管理员(admin) | 全部权限，包括系统配置、用户管理 | 技术负责人 |
| 编辑(editor) | 内容编辑、SEO优化、产品管理 | 内容运营 |
| 查看者(viewer) | 只读权限，可查看数据和报告 | 管理层/客户 |

### 2.3 修改密码

1. 点击右上角头像
2. 选择"个人设置"
3. 输入旧密码和新密码
4. 点击"保存"

## 3. SEO管理

### 3.1 SEO仪表盘

**路径**: SEO管理 > 仪表盘

仪表盘展示核心SEO指标：
- 关键词总数及排名分布
- AI搜索引擎收录情况
- 网站审计评分
- LLMs.txt状态

### 3.2 页面SEO管理

**路径**: SEO管理 > 页面SEO

#### 添加/编辑页面SEO

1. 点击"新增"按钮
2. 填写页面信息：
   - **页面URL**: 页面的完整URL
   - **Meta标题**: 建议50-60字符
   - **Meta描述**: 建议150-160字符
   - **关键词**: 多个关键词用逗号分隔
   - **Canonical URL**:  canonical链接（可选）
3. 点击"保存"

#### AI内容优化

1. 在页面列表中点击"AI优化"
2. 输入原始内容
3. 系统自动返回优化后的Meta标题和描述
4. 确认无误后点击"应用"

### 3.3 LLMs.txt生成器

**路径**: SEO管理 > LLMs.txt

LLMs.txt是面向AI搜索引擎的标准化文件，帮助AI理解网站内容。

#### 生成步骤

1. 填写业务信息：
   - 公司名称
   - 主营业务
   - 产品分类
   - 联系方式
2. 点击"生成LLMs.txt"
3. 预览生成内容
4. 点击下载或发布

#### LLMs.txt结构

```txt
# 公司信息
- 名称: 优丁建材有限公司
- 业务: 轻集料混凝土生产销售
- 成立: 20XX年

# 产品列表
- LC5.0轻集料混凝土
- LC7.5轻集料混凝土
- ...

# 技术参数
- 密度等级: LC5.0-LC25.0
- 强度等级: A型/B型
```

### 3.4 网站审计

**路径**: SEO管理 > 网站审计

#### 执行审计

1. 输入要审计的URL
2. 选择审计维度（共11项）：
   - Meta标签完整性
   -  heading结构
   - 图片alt属性
   - 内部链接
   - 外部链接
   - 移动端适配
   - 页面加载速度
   - SSL证书
   - Schema标记
   - EEAT信号
   - 内容质量
3. 点击"开始审计"
4. 等待审计完成（约1-3分钟）
5. 查看审计报告

#### 审计报告

报告包含：
- **综合评分**: 0-100分
- **问题列表**: 按严重程度分类
- **改进建议**: 具体修复方案

### 3.5 关键词排名追踪

**路径**: SEO管理 > 关键词排名

#### 导入关键词

1. 点击"导入关键词"
2. 选择方式：
   - **默认列表**: 导入预置的轻集料混凝土行业关键词
   - **自定义**: 粘贴关键词列表（每行一个）
3. 点击"确定"

#### 更新排名

1. 点击"更新排名"按钮
2. 系统自动查询各搜索引擎的排名
3. 等待更新完成
4. 查看排名变化

#### 查看历史

1. 在关键词列表中点击"历史"
2. 查看该关键词的排名趋势图
3. 可选择时间范围（7天/30天/90天）

### 3.6 Schema标记管理

**路径**: SEO管理 > Schema标记

Schema标记帮助搜索引擎理解页面内容，提升搜索结果展示效果。

#### 生成Schema

1. 选择Schema类型：
   - Product（产品）
   - Article（文章）
   - FAQPage（常见问题）
   - Organization（组织）
   - LocalBusiness（本地商家）
   - BreadcrumbList（面包屑）
2. 填写对应字段
3. 点击"生成JSON-LD"
4. 复制代码到页面<head>中

### 3.7 EEAT评分系统

**路径**: SEO管理 > EEAT

EEAT（Experience, Expertise, Authoritativeness, Trustworthiness）是Google评估内容质量的核心标准。

#### 作者管理

1. 点击"添加作者"
2. 填写作者信息：
   - 姓名
   - 职位
   - 从业年限
   - 资质证书
   - 社交媒体链接
3. 上传作者头像
4. 点击"保存"

#### EEAT评分

系统自动从四个维度评分：
- **经验(20%)**: 从业年限、案例数量
- **专业(25%)**: 认证资质、教育背景
- **权威(25%)**: 媒体引用、行业奖项
- **可信(30%)**: 信任信号、透明度

## 7. 常见问题

### Q1: 忘记管理员密码怎么办？

**A**: 联系技术人员通过数据库重置：
```sql
UPDATE users SET password_hash = '新密码哈希' WHERE username = 'admin';
```

### Q2: LLMs.txt在哪里访问？

**A**: 生成后会自动发布到 `https://youding.com/llms.txt`

### Q3: 关键词排名多久更新一次？

**A**: 建议每日更新一次，避免频繁请求被搜索引擎限制。

### Q4: 网站审计评分低怎么办？

**A**: 查看审计报告中的问题列表，按优先级逐一修复。常见问题：
- Meta标签缺失
- 图片没有alt属性
- 页面加载过慢
- 缺少Schema标记

### Q5: 如何备份数据？

**A**: 系统每天自动备份数据库，备份文件保存在 `/backups` 目录。也可手动执行：
```bash
docker compose exec postgres pg_dump -U youding_prod youding_seo_prod > backup.sql
```

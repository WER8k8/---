# API Integration

API集成工具集。支持RESTful API设计、接口开发、API网关配置、接口文档编写等功能，帮助开发者构建和管理API服务。

## 核心功能
- RESTful API设计
- 接口开发
- API网关配置
- 接口文档编写
- API测试
- 接口监控

## 触发场景
- API开发
- 接口集成
- 服务对接
- API管理

## MCP工具集成
- claude-api：API调用
- mcp-builder：MCP服务
- webapp-testing：接口测试

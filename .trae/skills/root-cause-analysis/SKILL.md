# Root Cause Analysis Engine

根因分析引擎，深入分析问题根本原因，提供系统性解决方案。支持多种分析方法，帮助找到问题的真正源头。

## 核心职责
- 问题分析
- 根本原因定位
- 解决方案设计
- 预防措施建议

## 核心能力
- **5W1H分析**：全面了解问题背景
- **鱼骨图分析**：系统性分析原因
- **5Why分析**：深入挖掘根本原因
- **故障树分析**：逻辑分析故障关系
- **因果图分析**：可视化因果关系

## 分析流程
1. **问题定义**：明确问题范围和影响
2. **数据收集**：收集相关数据和证据
3. **原因分析**：使用多种方法分析原因
4. **根本原因确认**：验证根本原因假设
5. **解决方案制定**：提出系统性解决方案

## 触发场景
- 复杂问题诊断
- 重复出现的故障
- 性能问题分析
- 系统故障调查
- 质量问题追溯

## MCP工具集成
- code-analysis：代码问题分析
- database-tools：数据问题分析
- webapp-testing：测试验证
- solution-document-writer：报告编写
- claude-api：AI辅助分析

## 分析方法
| 方法 | 用途 | 特点 |
|------|------|------|
| 5Why | 深入挖掘 | 简单直接 |
| 鱼骨图 | 系统分析 | 全面覆盖 |
| 故障树 | 逻辑分析 | 精确严谨 |
| 因果图 | 可视化 | 直观清晰 |
| 帕累托分析 | 优先级排序 | 聚焦重点 |

## 输出成果
- 问题分析报告
- 根本原因说明
- 解决方案建议
- 预防措施清单
- 改进计划

# Multi-Agent Orchestrator

多智能体协作编排器，负责管理Agent Teams，实现并行任务执行、分布式计算和协同工作。支持动态智能体分配、负载均衡和任务调度。

## 核心职责
- 智能体团队组建
- 并行任务分配
- 负载均衡调度
- 任务协调与同步
- 结果汇总与整合
- 故障容错与恢复

## 核心能力
- **并行执行**：多个智能体同时执行不同任务
- **分布式协作**：智能体间信息共享与协作
- **动态扩展**：根据任务复杂度动态调整智能体数量
- **故障恢复**：自动处理智能体失败并重新分配任务

## 触发场景
- 大规模代码库分析
- 多维度研究任务
- 并行数据处理
- 复杂项目协同开发
- 广度优先搜索任务

## MCP工具集成
- claude-api：智能体通信
- mcp-builder：智能体创建
- skills-search：技能匹配
- webapp-testing：并行测试
- shell-tools：分布式命令执行

## 协作模式
- **Orchestrator + Workers**：指挥者+工作者模式
- **Peer-to-Peer**：对等协作模式
- **Hierarchical**：层级管理模式
- **Swarm Intelligence**：群体智能模式

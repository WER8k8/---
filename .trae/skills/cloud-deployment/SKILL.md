# Cloud Deployment

云部署工具集。支持Docker容器化、Kubernetes编排、CI/CD流水线、云原生架构部署等功能，帮助开发者实现自动化部署和运维。

## 核心功能
- Docker容器化
- Kubernetes编排
- CI/CD流水线
- 云原生架构
- 自动化部署
- 资源管理

## 触发场景
- 容器部署
- 持续集成
- 云运维
- 基础设施管理

## MCP工具集成
- insulation-devops-engineer：DevOps支持
- solution-document-writer：部署文档
- webapp-testing：部署验证

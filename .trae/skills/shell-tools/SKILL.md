# Shell Tools

文件系统操作与命令执行工具集。支持文件读写、目录浏览、命令执行、文本搜索等功能，帮助开发者高效管理项目文件。

## 核心功能
- 文件读取/写入
- 目录浏览
- 命令执行
- 文本搜索（grep）
- Glob模式匹配
- 文件编辑

## 触发场景
- 文件操作
- 目录管理
- 命令执行
- 文本搜索

## MCP工具集成
- git-expert：版本控制
- webapp-testing：测试执行
- solution-document-writer：文档生成

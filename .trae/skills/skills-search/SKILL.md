# Skills Search

智能技能搜索与发现服务。使用向量嵌入和语义相似度算法检索相关的Claude Agent Skills，实现渐进式披露架构，使专业技能能够被任何MCP兼容的AI应用程序访问。

## 核心功能
- 向量嵌入搜索
- 语义相似度匹配
- 技能发现与推荐
- 渐进式披露架构支持
- 跨技能协作

## 触发场景
- 技能发现
- 工具推荐
- 能力匹配
- 知识库检索

## MCP工具集成
- mcp-builder：MCP服务集成
- doc-coauthoring：技能文档
- claude-api：AI辅助搜索
